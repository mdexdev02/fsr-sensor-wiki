## 측정 회로 구성

### 기본 회로도

```
      FSR Sensor
          │
    ┌─────┴─────┐
    │           │
   VCC         ADC
    │           │
    └─────┬─────┘
          │
         Rc
          │
         GND
```

- **Rc (풀다운 저항)**: 1KΩ ~ 47KΩ (권장: 1KΩ)
- **출력 전압**: Vout = Vin × (Rc / (Ra + Rc))
  - Ra: FSR 센서의 저항값
  - Rc: 레퍼런스 풀다운 저항

### Arduino 연결 예제

```cpp
// FSR 센서 읽기 예제
int fsrPin = A0;     // FSR 센서 연결 핀
int fsrReading;      // 센서 읽기 값

void setup() {
  Serial.begin(9600);
}

void loop() {
  fsrReading = analogRead(fsrPin);
  
  Serial.print("FSR Reading: ");
  Serial.println(fsrReading);
  
  // 압력 레벨 판단
  if (fsrReading < 10) {
    Serial.println(" - 압력 없음");
  } else if (fsrReading < 200) {
    Serial.println(" - 가벼운 터치");
  } else if (fsrReading < 500) {
    Serial.println(" - 중간 압력");
  } else if (fsrReading < 800) {
    Serial.println(" - 강한 압력");
  } else {
    Serial.println(" - 매우 강한 압력");
  }
  
  delay(100);
}
```
