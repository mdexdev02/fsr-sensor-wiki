# 기계적 특성

FSR 센서의 기계적 특성은 센서 선택과 적용 설계에 중요한 요소입니다.

| **RA9P** | 원형 | Ø9mm | Ø14mm | 0.45mm | 0.2g |

| **RA18** | 원형 | Ø18mm | Ø25mm | 0.45mm | 0.5g |
| **RA30P** | 원형 | Ø30mm | Ø38mm | 0.45mm | 0.8g |
| **QA3040P** | 사각형 | 30×40mm | 35×45mm | 0.45mm | 1.2g |
| **MDXS-16** | 방석형 | 400×400mm | 410×410mm | 1.5mm | 150g |

```
┌─────────────────────┐ ← 보호 필름 (선택)
```

### 압력 단위 변환

```javascript
// 단위 변환 함수
class PressureConverter {
  // Newton to kg-force
  static NtoKgf(newton) {
    return newton / 9.81;
  }
  
  // Newton to Pound-force
  static NtoLbf(newton) {
    return newton * 0.2248;
  }
  
  // Pressure (N/cm²) to PSI
  static NcmToPsi(ncm) {
    return ncm * 1.450;
  }
  
  // 센서 면적 기준 압력 계산
  static forceToPressure(force, sensorDiameter) {
    const area = Math.PI * Math.pow(sensorDiameter/2, 2);
    return force / area; // N/mm²
  }
}
```

### 최소 굴곡 반경

| **강성형** | 10mm | 20mm | 평면 설치 권장 |

### 굴곡 시 특성 변화

```cpp
// 굴곡 보상 알고리즘
float bendingCompensation(float resistance, float bendRadius) {
  const float FLAT_RADIUS = 1000.0; // 평면 = 매우 큰 반경
  const float BEND_FACTOR = 0.1;    // 굴곡 영향 계수
  
  float bendEffect = 1.0 + BEND_FACTOR * (1.0 - bendRadius/FLAT_RADIUS);
  return resistance / bendEffect;
}
```

## 설치 및 장착

### 장착 방법

#### 1. 양면 테이프
```
장점: 간단, 빠른 설치
단점: 영구 고정 어려움
용도: 프로토타입, 임시 설치
```

#### 2. 접착제
```
장점: 영구 고정, 얇은 두께
단점: 제거 어려움
용도: 최종 제품
```

#### 3. 기계적 고정
```
장점: 교체 가능, 확실한 고정
단점: 추가 부품 필요
용도: 산업용 응용
```

### 설치 시 주의사항

:::warning 중요
1. **평평한 표면**: 센서 아래 요철이 없어야 함
2. **균일한 압력**: 국부적 압력 집중 방지
3. **여유 공간**: 리드선 연결부 스트레스 방지
4. **절연**: 금속 표면과 직접 접촉 금지
:::

## 힘 분포 및 감도

### 감지 영역 내 감도 분포

```
감도 맵 (원형 센서 예시)
    높음 ← → 낮음
    ┌─────────┐
    │  ░░░░░  │
    │ ░█████░ │
    │░███████░│
    │░███████░│
    │░███████░│
    │ ░█████░ │
    │  ░░░░░  │
    └─────────┘
    중심: 100% 감도
    가장자리: 50% 감도
```

### 압력 분포 균일화

```cpp
// 압력 분포 보정
class PressureDistribution {
  private:
    float sensitivityMap[10][10]; // 감도 맵
    
  public:
    void calibrateMap() {
      // 중심부 높은 감도
      for(int i = 0; i < 10; i++) {
        for(int j = 0; j < 10; j++) {
          float dist = sqrt(pow(i-5, 2) + pow(j-5, 2));
          sensitivityMap[i][j] = 1.0 - (dist / 7.0) * 0.5;
        }
      }
    }
    
    float getCorrectedPressure(int x, int y, float raw) {
      return raw / sensitivityMap[x][y];
    }
};
```

## 재료 특성

### 기판 재료
- **PET (Polyester)**: 표준, 유연성 우수
- **PI (Polyimide)**: 고온 내성
- **PEN**: 고강도, 치수 안정성

### 전도성 재료
- **카본 잉크**: 표준, 저렴
- **은 잉크**: 고전도성, 고가
- **하이브리드**: 균형된 성능
