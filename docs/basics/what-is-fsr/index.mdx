# FSR 센서란?

## 개요

**FSR(Force Sensitive Resistor)**은 가해지는 힘이나 압력에 따라 저항값이 변화하는 센서입니다. 
압력이 증가하면 저항이 감소하는 특성을 가지고 있어, 간단한 회로로 압력을 측정할 수 있습니다.

:::info 핵심 특징
- 📏 얇고 유연한 형태 (두께 0.2~1.5mm)
- 💰 저렴한 가격
- 🔧 간단한 사용법
- ⚡ 낮은 전력 소비
:::

## 기본 구조

FSR 센서는 크게 두 개의 층으로 구성되어 있습니다:

### 상판 (Top Layer)
- **유연한 폴리에스터 필름**: 압력을 전달하는 역할
- **전도성 트레이스**: 전기 신호를 전달하는 경로
- **접착층**: 하판과의 결합을 위한 층

### 하판 (Bottom Layer)
- **전도성 폴리머 잉크**: 압력에 따라 저항이 변하는 핵심 소재
- **기판**: 센서의 물리적 지지대 역할
- **전극 패턴**: 전기적 연결을 위한 구조

```
┌─────────────────────┐  ← 압력 인가
│   상판 (Top Layer)  │
│  [유연한 필름]      │
├─────────────────────┤
│   스페이서/접착층    │
├─────────────────────┤
│   하판 (Bottom)     │
│  [전도성 폴리머]    │
└─────────────────────┘
```

## 작동 원리 요약

<Tabs>
  <TabItem value="no-pressure" label="압력 없음" default>

    - **저항**: 높음 (100kΩ ~ 1MΩ)
    - **출력**: 낮은 신호 레벨
  </TabItem>
  <TabItem value="high-pressure" label="높은 압력">
    - **상태**: 많은 전도성 입자들이 접촉
    - **저항**: 낮음 (1kΩ ~ 10kΩ)
    - **출력**: 높은 신호 레벨
  </TabItem>
</Tabs>

## 장단점 비교

### ✅ 장점

| 특성 | 설명 |
|------|------|
| **저렴한 가격** | 로드셀 대비 1/10 ~ 1/100 수준 |
| **얇고 유연함** | 0.2~1.5mm 두께로 좁은 공간에 설치 가능 |
| **간단한 사용** | 복잡한 신호 처리 회로 불필요 |
| **내구성** | 100만 회 이상 작동 가능 |
| **다양한 크기** | 5mm ~ 600mm까지 다양한 크기 제공 |

### ⚠️ 단점

| 특성 | 설명 | 해결 방법 |
|------|------|-----------|
| **정밀도 한계** | ±5~10% 오차 | 교정을 통한 보정 |
| **드리프트 현상** | 장시간 압력 시 값 변화 | 주기적 재교정 |
| **온도 영향** | 온도에 따른 저항 변화 | 온도 보상 회로 |
| **히스테리시스** | 압력 증가/감소 시 다른 값 | 소프트웨어 보정 |

## FSR vs 다른 압력 센서

| 특성 | FSR | 로드셀 | 압전 센서 |
|------|-----|--------|-----------|
| 정확도 | ±5\~10% | ±0.1\~1% | ±1\~3% |
| 가격 | 💰 낮음 | 💰💰💰 높음 | 💰💰 중간 |
| 두께 | 0.2\~1.5mm | 10mm 이상 | 1\~5mm |
| 응답속도 | 1\~2ms | 10ms | μs 단위 |
| 적용분야 | 일반 측정 | 정밀 측정 | 동적 측정 |

## 주요 응용 분야 미리보기

FSR 센서는 다양한 분야에서 활용됩니다:

- 🏥 **의료**: 재활 장비, 자세 모니터링
- 🏭 **산업**: 품질 검사, 로봇 그리퍼
- 🎮 **엔터테인먼트**: 게임 컨트롤러, 전자 악기
- 🎓 **교육**: STEAM 교육, 실험 장비
